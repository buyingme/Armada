# 0. Documentation Overview

## About this Documentation

This architecture documentation follows the [arc42 template](https://arc42.org/) (Version 8.2).
It describes the software architecture of the **Star Wars: Armada Digital Edition**, a computer game adaptation of the tabletop miniatures game *Star Wars: Armada*.

## How to Read

- **New to the project?** Start with [01 Introduction and Goals](01_introduction_and_goals.md) and [04 Solution Strategy](04_solution_strategy.md).
- **Looking for component details?** See [05 Building Block View](05_building_block_view.md).
- **Understanding runtime behavior?** Check [06 Runtime View](06_runtime_view.md).
- **Checking quality?** See [10 Quality Requirements](10_quality_requirements.md).

## Conventions

- Architecture Decision Records (ADRs) are maintained in [09 Architectural Decisions](09_architectural_decisions.md).
- Diagrams are stored in the `images/` subfolder and referenced inline.
- This documentation is a living document — update it as the architecture evolves.
