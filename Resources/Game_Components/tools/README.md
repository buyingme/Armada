# tools/

Measurement tool images (range ruler).

## Naming Convention

`range_ruler_<type>.png`

| File | Description |
|------|-------------|
| `range_ruler_range.png` | Range bands — Close / Medium / Long |
| `range_ruler_distance.png` | Distance measurement side |
