Card Text
" Icon Command Repair: Gain 1 additional engineering point."

Rules Clarification
A ship gains an additional engineering point for every Icon Command Repair command.

When a Icon Command Repair token is spent, you half the engineering value first and add 1 point second.
If Icon Command Repair token and Icon Command Repair dial are spent together, that's a single command. You only gain 1 additional engineering point.
