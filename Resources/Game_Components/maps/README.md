# maps/

Background map images for the play area.

Map size is authoritative from the filename prefix:

- `map_3x3...` files are 3 ft x 3 ft play-area maps.
- `map_3x6...` files are 3 ft x 6 ft play-area maps.

The fleet builder filters these pools separately. Core Set 180-point rosters use
3x3 maps; 300- and 400-point rosters use 3x6 maps.

## Naming Convention

`map_<grid>_<name>_v<version>.jpg`
