Card Text
" Icon Command Navigate: Your Icon Command Navigate tokens can either change your speed or increase your yaw value by 1."

Rules Clarification
Nav Team’s effect is not cumulative with General Madine’s effect.

A friendly ship equipped with Nav Team that resolves a Icon Command Navigate command using a Icon Command Navigate token to increase a yaw value may resolve Admiral Ozzel’s effect to change its speed by 1.
