# dice/

Attack dice face images. 12 PNGs total (4 per colour, 3 colours).

## Naming Convention

`die_<colour>_<face>.png`

| File | Die | Face |
|------|-----|------|
| `die_red_blank.png` | Red | Blank |
| `die_red_hit.png` | Red | Hit |
| `die_red_hit_hit.png` | Red | Double-Hit |
| `die_red_crit.png` | Red | Critical |
| `die_blue_blank.png` | Blue | Blank |
| `die_blue_hit.png` | Blue | Hit |
| `die_blue_accuracy.png` | Blue | Accuracy |
| `die_blue_hit_crit.png` | Blue | Hit-Critical |
| `die_black_blank.png` | Black | Blank |
| `die_black_hit.png` | Black | Hit |
| `die_black_crit.png` | Black | Critical |
| `die_black_hit_hit.png` | Black | Double-Hit |
